⚡ ACTIVAR BASE DE DATOS EN LA NUBE (Firebase)
Paso 1: Entrá a console.firebase.google.com
Paso 2: Creá un proyecto nuevo (nombre libre, desactivá Google Analytics)
Paso 3: En el panel izquierdo → Realtime Database → "Crear base de datos" → modo prueba
Paso 4: Volvé al inicio del proyecto → "Agregar app web" (</>) → copiá el objeto firebaseConfig
Paso 5: Abrí el index.html en un editor de texto → buscá FIREBASE_CONFIG (arriba del todo en el JS) → pegá tus datos
Paso 6: Guardá y listo. Los datos quedan en la nube para siempre 🚀
ℹ️ Mientras no configures Firebase, los datos se guardan localmente en este navegador (funciona igual, pero solo en este dispositivo).
